To test, start (one of) the server(s):

: % ./http-daemon-with-timeout

Given some longish request (give or take a few characters), the server will
hang. Make it shorter and it won't anymore.

: % curl -vvv -k 'https://localhost:8084/api/account.shared.modify/create_account?a=--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------'

Tested with HTTP::Daemon::SSL 1.04 (won't hang for long requests if Timeout
option is not given) and HTTP::Daemon::SSL 1.05_01 (hang for long requests for
both http-daemon-with{,out}-timeout).
